自己紹介

こんにちは、菊地泰史です。現在中学生で、将来はAI開発者になることを目指しています。Pythonを中心にプログラミングを学習し、仮想環境の構築や機械学習の基本的なアルゴリズムの実装に取り組んでいます。

また、ゲーム開発にも強い関心があり、UnityやUnreal Engineを使ったゲーム制作の経験も積んでいます。

趣味は天体観測と天体撮影で、望遠鏡を使って月や惑星の観測を行いています。

スキル

  • Python / AI 開発: scikit-learn を用いた機械学習、データ前処理、クロスバリデーションの実装経験
  • HTML / CSS / JavaScript: 個人ブログやポートフォリオサイトの制作、レスポンシブデザイン対応
  • Unity / Unreal Engine: 2D・3Dゲームの制作、ブループリントによるイベント駆動型設計
  • その他: Gitを利用したバージョン管理、GarageBandを用いた音楽制作

AIプロジェクト

Pythonを使って機械学習モデルを構築し、簡単な画像分類アプリを作成しました。データの前処理からモデル評価まで一連の流れを経験しています。

ゲーム開発

UnityとUnreal Engineを使ってRPG風のゲームを制作しました。キャラクターの移動、戦闘システム、UI設計などを担当しました。

天体撮影

望遠鏡とカメラを用いて月や土星を撮影し、画像処理ソフトで編集しました。天体写真コンテストに応募した経験もあります。
